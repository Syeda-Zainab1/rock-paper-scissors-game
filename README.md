## ✨ Features

- **Score Tracking:** Keeps track of both User and Computer scores during the session.
- **Smart Logic:** Uses JavaScript to generate random computer moves.
- **Visual Feedback:** The message bar changes color based on whether you win, lose, or draw.
- **Responsive Design:** Clean layout with interactive hover effects.

## 📂 Project Structure

- `index.html`: The game interface and score board.
- `game.css`: Dark-themed styling and circular choice buttons.
- `game.js`: The game engine, including random choice generation and win logic.

## 🛠️ How to Play

1. Click on one of the three images (Rock, Paper, or Scissors).
2. The computer will immediately make its choice.
3. The message bar will update to tell you the result, and the score board will update automatically!
